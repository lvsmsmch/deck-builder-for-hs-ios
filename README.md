# Deck Builder for Hearthstone iOS

Native SwiftUI iOS version of the Hearthstone deck builder.

## Features

- Card library backed by HearthstoneJSON.
- Search, mana, class, format, rarity, and type filters.
- Card detail screen with official HearthstoneJSON card renders.
- Saved decks with local JSON persistence.
- Deck code import, export, copy, and share.
- Native deck builder with class picker, card pool, deck list, Standard/Wild formats, and singleton mode.
- Theme and card-language settings.

## Tech

- SwiftUI
- async/await
- URLSession
- Codable
- UserDefaults and local JSON storage

No third-party packages are required.
